# 🐾 VetOnlineCRM

CRM-система для ветеринарных онлайн-консультаций.

## Возможности

- 👥 Управление владельцами и питомцами
- 💊 Ведение приёмов с рекомендациями
- 📝 Шаблоны рекомендаций с категориями
- 📅 Календарь записей на приём
- 🔔 Напоминания
- 📋 Опросники для клиентов (публичные ссылки)
- 📄 Экспорт в PDF и Word
- 📎 Загрузка файлов
- 🌓 Светлая/тёмная тема
- 👨‍⚕️ Мультиврач (админ-панель)

## Установка

```bash
# Клонируем
git clone https://github.com/YOUR_USERNAME/vet-online-crm.git
cd vet-online-crm

# Виртуальное окружение
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# Зависимости
pip install -r requirements.txt

# Запуск
python run.py
